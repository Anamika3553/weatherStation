🌦️ Weather Station

A simple and responsive Weather Station web application that allows users to check the current weather conditions of any city using the OpenWeatherMap API.

📌 Features

- 🌍 Search weather by city name
- 🌡️ Displays current temperature in Celsius
- ☁️ Shows weather icon based on current conditions
- ⚠️ Handles invalid city names with an alert
- 📱 Responsive design using Bootstrap
